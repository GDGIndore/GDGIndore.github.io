Docker for Develpopers

Docker is way ahead of Virtualization. Docker containers wrap up a piece of software in a complete filesystem that contains everything it needs to run: code, runtime, system tools, system libraries – anything you can install on a server. This guarantees that it will always run the same, regardless of the environment it is running in. This talk/session will focus on how to use this simple tool to create containers for our testing and development environments.

Open Stack

OpenStack is an open source cloud computing framework, for creating public and private clouds. It controls large pools of compute, storage, and networking resources throughout a datacenter, managed through a dashboard, CLI or via the OpenStack API.
This talk aims to give the audience as much information as possible about what/why OpenStack, its origin, architecture and details of its main components.
